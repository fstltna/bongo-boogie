La boite de bienvenue est le premier écran visible au lancement du jeu.

Appuyez sur HELP ( AIDE si vous selectionnez le français ) vous emmera vers l'aide du jeu.

Le bouton FULLSCREEN ( Plein Ecran ) vous permettra de lancer le jeu en plein écran offrant un meilleur confort de jeu.

Cliquez sur Parental Lock ( Controle parental ) pour activer/désactiver tout contenu pouvant choquer la sensibilité des plus petits.

Play ( Jouer ) démarre le jeu.

QUIT ( Quitter ) Retourne immediatement au systeme d'exploitation.

Cliquez sur la liste des langages pour selectionner le langage de votre choix.
