Foire aux questions

Le jeu ne démarre pas.

Check Verifiez que votre carte vidéo est équipée de 8 Mo de Mémoire ( ou plus )

Le jeu ne tourne pas sur les cartes videos S3 Savage 4 ( Celles ci ne sont pas équipées d'un accelerateur directdraw ).

Peut on changer le mot de passe du controle parental ?

Oui il faut remplacer le texte du fichier plock.txt qui se situe dans le repertoire de l'application

Exemple :

parent
par
votremotdepasse

Comment réinitialiser les scores ?

Allez dans les options et cliquez sur "Reset Scores"

Le jeu est trés lent quand je joue dans une fenetre !

Essayez de reduire la prodondeur d'écran a 16 Bits par pixel, si le jeu est toujours trop lent, essayez de jouer en plein écran.
Si le jeu est encore trop lent, pensez a remplacer votre carte vidéo par un modéle plus récent.
