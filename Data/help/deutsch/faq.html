Frequently Asked Questions

Das Spiele kann nicht gestartet werden!

Sie brauchen mindestens 8MB Grafikkartespeicher. Stellen Sie sicher das alle Gerätetreiber installiert sind.
Das Spiel läuft nicht mit S3 Savage 4 video Karten ( Diese Karten haben keine directdraw Beschleunigung )

Kann ich das Kennwort für die elternliche Kontrolle ändern?

Sie können die plock.txt Datei in dem Spiel Ordner ändern. Ändern Sie das Feld:
parent
mit einen neuen
Kennwort

Kann ich den Punktestand zurücksetzen

Öffnen Sie das Optionfenster und klicken Sie die Punkte-Zurückstellen Taste.

Das Spiel ist zulangsam wenn ich es in einem Fenster spiele!.

Reduzieren Sie ihre Bildschirmauflösung zu 16bits/pixel. Sollte das nicht helfen, spielen Sie das Spiel in Fullscreen, das wird die Spieleleistung erheblich erhöhen.
Sollte das Spiel immer noch zulangsm sein, müßen Sie eine schneller Graphikkarte installieren.

Ich habe meinen Registration Code verloren!

Senden Sie uns eine Email mit Ihren Kontakt Informationen und wir werden ihnen Ihren Registration Code zuschicken.
